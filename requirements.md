# 📋 Frontend Exercise

## 🎯 What is the problem?

- Our clients need a place where they can log in
with their username and password to view their accounts.

- We need administrator access where all
users and their accounts can be viewed.

------------------------------------------------------------------------

## 🛠️ What should I do?

- Create a frontend web application using your preferred library or framework:

- jQuery, Angular, React, Vue, Backbone, etc.

------------------------------------------------------------------------

## 🔐 Authentication

- The application must include:

- Login for **users**

- Login for **administrators**

- You must use **JWT (JSON Web Tokens)** to protect the
private sections.

- Please note:

- Some users are **disabled** and should return an error message when attempting to log in.

------------------------------------------------------------------------

## 👨‍💼 Administrator Features

- View a **client list**
- For each client:

- View their **transaction history**

------------------------------------------------------------------------

## 👤 Client Features

- View their own **transaction history**

------------------------------------------------------------------------

## 📊 Data Management

All lists must include:

- **Pagination**
- **Sort by any column**
